# SakuraServerMod
## 简介
一个适用于自己服务器的模组,增加了一些特性以保证生存舒适度
## 特性
### 附魔:
#### 灵魂绑定
死亡时,该物品不会掉落.无法通过附魔台获得.  
是宝藏附魔,最大等级:1.  
#### 连锁挖矿:
默认快捷键:`或~  
顾名思义,一键挖取矿物,挖取的个数为 16 * 等级.  
使用会额外增加疲劳度,增加的疲劳度为: 0.7 - 等级 * 0.1 /方块  
非宝藏附魔,最大等级:4.
### 指令:
#### /removelore
清除身上所有物品的lore信息.
### 其他修改:
1. 使无限与经验修补兼容.
2. 去除铁砧在生存模式的40级上限.
